Bloc Caractéristiques sociales supplémentaires
operationalSocialFeaturesDetail identifyingSocialFeaturesDetail

Superclasse / Générique

Description:
Données supplémentaires sur les modalités de mise en œuvre de la signalétique sociale.

Présence:
Obligatoire si au moins une des sous-classes / spécialisations doit être communiquée. Ne peut pas être mentionné seul.

Nom du champ Description TechLabel
Date de début La date de début de la période dans la période de la signalétique sociale ou dans la période de la signalétique sociale périodique à partir de laquelle le set de données, constituant une des sous-classes des caractéristiques sociales supplémentaires, est valable.

Si une ou plusieurs donnée(s) change(nt) dans une des sous-classes des  caractéristiques sociales supplémentaires, la date de début du bloc correspond à la date à partir de laquelle le nouveau set de données est d'application. 		startDate
Date de fin La date de fin de la période dans la période de la signalétique sociale ou dans la période de la signalétique sociale périodique jusqu'à laquelle le set de données, constituant une des sous-classes des caractéristiques sociales supplémentaires, est valable.

Si une ou plusieurs donnée(s) change(nt) dans une des sous-classes des caractéristiques sociales supplémentaires, la date de fin du bloc correspond à la date jusqu'à laquelle l'ancien set de données est d'application. 		endDate
Référence du déclarant Référence utilisée par le déclarant. externalReference
