Bloc Élément financier
financialElements

Description:
Eléments financiers attribués à la personne physique par l'entreprise.
Il s'agit d'un bloc générique pouvant contenir différents éléments financiers comme les rémunérations, les cotisations de sécurité sociale, les réductions de cotisations de sécurité sociales, ...

Présence:
Obligatoire, si un élément financier a été attribué à la personne physique

Nom du champ Description TechLabel
Code élément financier Code qui identifie l'élément financier spécifique. code
Montant Le montant de l'élément financier. amount
Rôle de l'élément financier Rôle de l'élément financier dans le calcul salarial. roles
Référence du déclarant Référence utilisée par le déclarant. externalReference
