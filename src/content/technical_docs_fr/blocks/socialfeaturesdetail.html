Blok Caractéristiques sociales supplémentaires
socialFeaturesDetail

Superclasse / Générique

Données supplémentaires sur les modalités de mise en œuvre de la signalétique sociale.

Nom du champ Description TechLabel
Numéro de suivi Numéro de suivi du bloc "caractéristiques sociales supplémentaires" dans le bloc "signalétique sociale" ou dans le bloc "signalétique sociale périodique". Ce numéro doit être unique dans le bloc situé au-dessus. sequenceNumber
Date de début La date de début de la période dans la période de la signalétique sociale ou dans la période de la signalétique sociale périodique à partir de laquelle le set de données constituant le bloc « caractéristiques sociales supplémentaires » s'applique. Si une ou plusieurs donnée(s) change(nt) dans le bloc « caractéristiques sociales supplémentaires », la date de début du bloc « caractéristiques sociales supplémentaires » correspond à la date à partir de laquelle le nouveau set de données est d'application. startDate
Date de fin La date de fin de la période dans la période de la signalétique sociale ou dans la période de la signalétique sociale périodique jusqu'à laquelle le set de données constituant le bloc « caractéristiques sociales supplémentaires » est valable. Si une ou plusieurs donnée(s) change(nt) dans le bloc « caractéristiques sociales supplémentaires », la date de fin du bloc « caractéristiques sociales supplémentaires » correspond à la date jusqu'à laquelle l'ancien set de données est d'application. endDate
