Notion partenaire moins valide

Indication si le partenaire de la personne physique est considéré comme une personne avec moins valide sur le plan fiscal, comme appliqué dans le cadre du calcul salarial.

TechLabel
disabilityOfPartner
Type
integer
Longeur
2
Format
-
Domaine de définition
1 = le partenaire n'est pas considéré sur le plan fiscal comme personne moins valide 2 = le partenaire est considéré sur le plan fiscal comme personne moins valide
Présence
Obligatoire si la personne physique est mariée ou cohabitant légal.
