Rôle de l'élément financier

Description
Rôle de l'élément financier dans le calcul salarial.
TechLabel
roles
Type
Lijst van integers
Longeur
2
Format
-
Domaine de définition
1 = montant qui fait effectivement partie du montant brut soumis aux retenues ONSS
2 = montant qui fait fictivement partie du montant brut soumis aux retenues ONSS
3 = retenue personnelle ONSS
4 = diminution d'une retenue personnelle ONSS
5 = mouvement positif sur le montant brut imposable
6 = mouvement négatif sur le montant brut imposable
7 = retenue fiscale
8 = diminution d'une retenue fiscale
9 = mouvement positif sur le montant net
10 = mouvement négatif sur le montant net
11 = renseignement
Présence
Indispensable
