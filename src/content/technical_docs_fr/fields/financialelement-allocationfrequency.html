Fréquence d'attribution

La fréquence en mois à laquelle un élément financier est accordé.

TechLabel
allocationFrequency
Type
integer
Longeur
2
Format
-
Domaine de définition
[0; 99] Exemples : Une fois par mois = 1 Une fois par trimestre = 3 Une fois par an = 12 Une fois tous les deux ans = 24 Pas de périodicité = 0
Présence
Obligatoire s’il est indiqué dans in que cette zone doit être remplie pour le code financier en question.
