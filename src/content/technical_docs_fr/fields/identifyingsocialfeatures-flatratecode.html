Code forfait

Description
Le code forfait détermine, sur base de la fonction de la personne, la rémunération journalière forfaitaire utilisée pour le calcul des cotisations de sécurité sociale.
TechLabel
flatRateCode
Type
integer
Longeur
5
Format
-
Domaine de définition
Voir annexe "codes forfait".
Présence
Obligatoire si la cotisation de sécurité sociale est calculée sur la base d’un forfait.
