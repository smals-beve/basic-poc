Blok Financieel element
financialElements

Beschrijving:
De financiële elementen toegekend aan de natuurlijke persoon door de onderneming.
Het betreft een generiek blok dat verschillende financiële elementen kan omvatten, zoals bezoldigingen, socialezekerheidsbijdragen, socialezekerheidsbijdrageverminderingen, …

Aanwezigheid:
Verplicht, indien een financieel element is toegekend aan de natuurlijke persoon.

Veldnaam Beschrijving TechLabel
Code financieel element Code die het specifieke financieel element identificeert. code
Bedrag Het bedrag van het financieel element. amount
