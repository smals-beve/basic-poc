Blok Financieel element
financialElement

De financiële elementen toegekend aan de natuurlijke persoon door de onderneming. Het betreft een generiek blok dat verschillende financiële elementen kan omvatten, zoals bezoldigingen, socialezekerheidsbijdragen, socialezekerheidsbijdrageverminderingen, …

Veldnaam Beschrijving TechLabel
Volgnummer Volgnummer van het blok "financieel element" binnen een blok "sociale signalitiek", "periodieke sociale signalitiek" of "fiscale signalitiek". Dit nummer moet uniek zijn binnen het bovenliggende blok. sequenceNumber
Type financieel element Code die het type financieel element identificeert. financialType
Code financieel element Code die het specifieke financieel element identificeert. financialCode
Bedrag Het bedrag van het financieel element. Nog te verduidelijken: wanneer moeten positieve bedragen worden aangegeven en wanneer moeten negatieve bedragen worden aangegeven. amount
Toewijzingsfrequentie De frequentie in maanden waarmee een financieel element wordt toegewezen. allocationFrequency
Aantal De hoeveelheid van een financieel element. numberOfUnits
Bedrag per eenheid Het bedrag van één eenheid van een financieel element. amountPerUnit
Notie onderworpen aan socialezekerheidsbijdragen Aanduiding of het financieel element in kwestie al dan niet is onderworpen aan socialezekerheidsbijdragen subjectionToSocialSecurityContributions
