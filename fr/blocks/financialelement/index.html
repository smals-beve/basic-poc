Blok Élément financier
financialElement

Eléments financiers attribués à la personne physique par l'entreprise. Il s'agit d'un bloc générique qui contient différents éléments financiers comme les rémunérations, les cotisations de sécurité sociale, les réductions de cotisations de sécurité sociales, ...

Nom du champ Description TechLabel
Numéro de suivi Numéro de suivi du bloc "élément financier" dans un bloc "signalétique sociale", "signalétique sociale périodique" ou "signalétique fiscale". Ce numéro doit être unique dans les blocs situés au-dessus. sequenceNumber
Type d'élément financier Code qui identifie le type d'élément financier. financialType
Code élément financier Code qui identifie l'élément financier spécifique. financialCode
Montant Le montant de l'élément financier. Encore à clarifier : quand les montants positifs doivent être déclarés et quand les montants négatifs doivent être déclarés. amount
Fréquence d'attribution La fréquence en mois à laquelle un élément financier est accordé. allocationFrequency
Nombre La quantité d'un élément financier. numberOfUnits
Montant par unité Le montant d'une unité d'un élément financier. amountPerUnit
Notion d'assujettissement aux cotisations de sécurité sociale Indication si l'élément financier en question est assujetti ou non aux cotisations de sécurité sociale. subjectionToSocialSecurityContributions
